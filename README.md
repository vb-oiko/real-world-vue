# real-world-vue

## Project setup
```
yarn install
```

### json-server
You need to install json-server globally to imitate backend
```
npm i -g json-server
```

### Compiles and hot-reloads for development
```
json-server -d 1500 --watch db.json 
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
